# Ashley Hunter — Data Analyst Portfolio

Personal portfolio website built with HTML and CSS, hosted on GitHub Pages.

## About

This site showcases my work as a data analyst with 2+ years of experience in Power BI, SQL, Python, and data visualization. It includes my work history, technical skills, selected projects, and contact information.

## Structure

```
├── index.html       # Main portfolio page
├── style.css        # Styles and layout
└── README.md        # This file
```

## Tech Stack

- HTML5 / CSS3
- Google Fonts (DM Serif Display, Manrope)
- Vanilla JavaScript (scroll animations, smooth nav)
- Hosted via GitHub Pages

## Running Locally

Open in VSCode with the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension, then click **Go Live** in the status bar.
